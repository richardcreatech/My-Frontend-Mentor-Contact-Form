# Contact Form Challenge 🚀

A responsive contact form built with **React** as part of a Frontend Mentor challenge.

## Overview

This project is a solution to a Frontend Mentor challenge focused on building an **accessible and responsive contact form**.  
Users can enter their details, select a query type, and submit a message with **client-side validation** for a smooth and interactive experience.

## Features

- Responsive design for mobile, tablet, and desktop  
- Accessible form inputs and labels  
- Client-side form validation  
- Interactive UI built with React components
